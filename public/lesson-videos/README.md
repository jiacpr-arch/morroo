# วิดีโอประกอบบทเรียน (lesson-videos)

วิดีโอใส่ในบทเรียนได้ **2 แบบ** — เลือกตามความเหมาะสม

## แบบที่ 1 — ฝัง YouTube (แนะนำ: ง่าย ไม่กินพื้นที่ ไม่ต้องวางไฟล์)

ไม่ต้องวางไฟล์ในโฟลเดอร์นี้ ใช้แค่ **รหัสวิดีโอ** (ส่วนหลัง `v=` ใน URL)
เช่น `https://www.youtube.com/watch?v=dQw4w9WgXcQ` → รหัสคือ `dQw4w9WgXcQ`

แก้ `src/courses/firstaid/lessons.js`:

```js
// วิดีโอล้วน
{ type: 'video', heading: 'วิธีทำ CPR', youtube: 'dQw4w9WgXcQ',
  caption: 'สาธิตการกดหน้าอกที่ถูกต้อง' }

// วิดีโอใต้ข้อความ
{ type: 'read', heading: '...', body: '...',
  video: { youtube: 'dQw4w9WgXcQ', caption: 'ดูวิธีทำ' } }
```

> ฝังด้วย `youtube-nocookie.com` (โหมดความเป็นส่วนตัว) — ต้องมีอินเทอร์เน็ต ดูแบบออฟไลน์ไม่ได้

## แบบที่ 2 — ไฟล์วิดีโอที่อัปเอง (เล่นในแอปได้โดยตรง)

1. วางไฟล์ `.mp4` ไว้ในโฟลเดอร์นี้ (`public/lesson-videos/`) เช่น `cpr-demo.mp4`
2. อ้างอิงด้วย path `/lesson-videos/<ชื่อไฟล์>`

```js
{ type: 'video', heading: 'วิธีทำ CPR',
  src: '/lesson-videos/cpr-demo.mp4',
  poster: '/lesson-images/cpr-hand-position.svg',   // รูปปกก่อนเล่น (ไม่ใส่ก็ได้)
  caption: 'สาธิตการกดหน้าอกที่ถูกต้อง' }
```

## ข้อแนะนำสำหรับไฟล์ที่อัปเอง

- ใช้ `.mp4` (codec H.264 + AAC) เปิดได้ทุกเบราว์เซอร์
- บีบอัดให้ไฟล์เล็ก: ความละเอียด 720p ก็เพียงพอ
- ตั้งชื่อภาษาอังกฤษ ตัวเล็ก คั่นด้วย `-` เช่น `bandage-wrap.mp4`
- ใส่ `poster` (รูปปก) เพื่อให้ดูดีก่อนกดเล่น
- ไฟล์วิดีโอ **ไม่ถูกแคชออฟไลน์** โดยอัตโนมัติ (ไฟล์ใหญ่เกินขีดจำกัด PWA) — ต้องมีเน็ตตอนเล่น
