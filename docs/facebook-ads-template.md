# Facebook Ads Template — Traffic Campaign (ฟรี 30 วัน)

ใบยิง Facebook Ads สำหรับ MorRoo — โปรโมชั่น "ทดลองใช้ฟรี 30 วัน"
Objective: **Traffic** (ส่งคนเข้าเว็บเพื่อสมัครและทดลองทำข้อสอบ)

---

## 1. ภาพรวม Campaign

| รายการ | รายละเอียด |
|---|---|
| Objective | Traffic (Link Clicks / Landing Page Views) |
| Conversion Location | Website (https://morroo.com/register?utm_source=fb&utm_campaign=free30) |
| Optimization | Landing Page Views (ไม่ใช่ Link Clicks เพราะกรอง bot และคนรอโหลด) |
| Pixel Event ที่ track | PageView, ViewContent, CompleteRegistration, StartTrial |
| งบเริ่มต้นต่อวัน | ฿200–฿500 ต่อ ad set (ทดสอบ 3 ad set พร้อมกัน = ฿600–฿1,500/วัน) |
| ระยะเวลา test | 3 วันแรก = learning phase ห้ามแตะ, วันที่ 4 เริ่มตัด ad ที่ CPC สูง |
| Bid Strategy | Highest Volume (Lowest cost) — ไม่ใส่ cap ช่วง test |
| Schedule | 24/7 ช่วงแรก, เก็บข้อมูล 7 วันแล้วค่อย day-parting |

### KPI เป้าหมาย

| Metric | ดี | ปานกลาง | ต้องตัด |
|---|---|---|---|
| CPC (Link Click) | < ฿3 | ฿3–฿6 | > ฿6 |
| CTR (Link) | > 2% | 1–2% | < 1% |
| CPM | < ฿150 | ฿150–฿300 | > ฿300 |
| Cost per Registration | < ฿50 | ฿50–฿100 | > ฿100 |
| Hook Rate (3-sec view) | > 30% | 20–30% | < 20% |

## 2. Targeting (Ad Set)

### Ad Set A — Interest-based (กว้าง)
- **อายุ:** 20–35
- **เพศ:** ทุกเพศ
- **ภาษา:** ไทย
- **ที่อยู่:** ประเทศไทย
- **Interests:** แพทย์, นักศึกษาแพทย์, โรงพยาบาล, การแพทย์, ใบประกอบวิชาชีพเวชกรรม, ศิริราช, จุฬาฯ, รามาธิบดี
- **Behaviors:** Engaged Shoppers
- **Placements:** Advantage+ Placements (ให้ FB เลือกเอง)

### Ad Set B — Lookalike (ถ้ามี data)
- **Source:** Custom Audience จาก purchasers หรือ registered users (ขั้นต่ำ 100 คน)
- **Lookalike %:** 1% (similar audience)
- **ที่อยู่:** ประเทศไทย
- **อายุ:** 20–40

### Ad Set C — Broad (ไม่ใส่ interest)
- **อายุ:** 22–32
- **ที่อยู่:** ประเทศไทย
- **ภาษา:** ไทย
- **Interests:** ไม่ใส่ (ให้ algorithm ของ Meta หาเอง)
- **Placements:** Advantage+ Placements

> Tips: ตั้งแต่ปี 2024 Meta แนะนำ "broad targeting + ครีเอทีฟดี" มากกว่า narrow interest

## 3. Daily Creative Plan (30 วัน)

### Week 1 — Awareness & Hook Testing (วันที่ 1–7)
- ครีเอทีฟแนว "ปัญหา" — เน้น pain point ของผู้สอบ
- โพสต์รูป static 5 ตัว + วิดีโอสั้น 2 ตัว (15 วินาที)
- หมุนเวียน creative ทุก 2 วันถ้า CTR < 1%

### Week 2 — Social Proof (วันที่ 8–14)
- รีวิวจริงจากผู้สอบผ่าน (screenshot LINE/แชต)
- คลิป testimonial 30 วินาที
- before/after ผลคะแนน

### Week 3 — Feature Showcase (วันที่ 15–21)
- โชว์ฟีเจอร์: ข้อสอบจำลอง, AI วิเคราะห์จุดอ่อน, รายงานความก้าวหน้า
- Screen recording ของหน้าเว็บ (15 วินาที)
- Carousel 5 รูป — feature ละรูป

### Week 4 — Urgency & Retargeting (วันที่ 22–30)
- Retarget คนที่เข้าเว็บแต่ไม่สมัคร (Custom Audience: Website Visitors 14 days)
- เพิ่ม urgency: "ฟรี 30 วัน — จำกัด 100 สิทธิ์แรกของเดือน"
- Countdown visual

## 6. Tips การยิงจริง

1. **ใส่ text บนรูปนอก image model** — DALL·E สะกดไทยพลาดบ่อย ใช้ Canva/Figma เติมข้อความหลัง gen
2. **กฎ 20% text** — Facebook ไม่ reject แล้วแต่ algorithm ยัง favor รูปที่มี text น้อย
3. **ทำ 3 sizes ต่อ creative** — 1:1 (feed), 4:5 (feed mobile), 9:16 (story/reels)
4. **อย่าตัด ad ก่อน 3 วัน** — learning phase ต้องใช้ ~50 conversions
5. **CBO vs ABO** — เริ่มด้วย **ABO** (budget ที่ ad set) เพื่อ control test, พอเจอ winner ค่อยย้ายเป็น CBO
6. **Naming convention** — `[Objective]_[Audience]_[Creative]_[Date]` เช่น `Traffic_LLA1_Pain01_2026-05-13`
7. **UTM ทุกลิงก์** — `?utm_source=fb&utm_medium=paid&utm_campaign=free30&utm_content=pain01`
8. **Pixel + CAPI** — ต้องมีทั้งสองเพราะ iOS 14.5+ ลด accuracy ของ pixel อย่างเดียว
9. **Retargeting funnel** — Day 1–7: cold, Day 8+: retarget video viewers 50%+ และ website visitors 14d

## 8. Optimization Workflow รายสัปดาห์

**ทุกวันจันทร์ 9 โมง:**
1. ดู ad ที่ spend > ฿500 แต่ไม่มี conversion → ปิด
2. ดู ad ที่ CTR > 2% + CPC < ฿3 → duplicate ขึ้น budget 20%
3. รูป gen ใหม่ 3 ตัว จาก prompt ใหม่ → ใส่เป็น creative test
4. Audit pixel events ใน Events Manager — quality score ต้อง green
5. เช็ค frequency — ถ้า > 3 ต่อ ad set แสดงว่า audience เริ่มล้า ต้องขยายหรือเปลี่ยน creative
